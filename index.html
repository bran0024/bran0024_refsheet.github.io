Table

HTML and CSS reference sheet

Table Tags

Tag Name Description
<table> Table Creates a table
<thead> Table head Sections the head
<tbody> Table body Sections the main body
<tfoot> Table footer Sections the final part of the table. Usually for conclusive data like "Total".
<th> Table header To represent rows and columns.
<tr> Table row Add a row to the table

Table Attributes

Attributes Name Description
scope To indicate if it's a column or row. Usually for screen readers.
colspan Column span The number of columns that the data is taking up
rowspan Row span The number of rows that the data is taking up
